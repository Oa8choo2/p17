# On markdown

Markdown was created by John Gruber in 2004 (with help from Aaron
Swartz) to solve the problem of writing for the web without the
complexity of HTML.

It looks like this:

```
a piece of **markdown** found at [docs.hpc](docs.hpc.ugent.be)
```

which is rendered like this:

a piece of **markdown** found at [docs.hpc](docs.hpc.ugent.be)
