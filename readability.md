# readability

## Readability via vale plugin

Vale can be used to measure [readability of
texts](https://readable.com/readability). Tests are

-   *Automated Readability Index* (see next slide)
-   Coleman–Liau Index grade
-   Flesch reading ease score
-   Flesch–Kincaid grade level
-   Gunning-Fog index
-   LIX score
-   SMOG grade

## Automated Readability Index with *vale*

This is a test advised for technical texts. A high score indicates a
complex text. Range is 5-22.

Our texts score below 12. This should be ok for *our* customers.

See

-   [Automated readability index -
    Wikipedia](https://en.wikipedia.org/wiki/Automated_readability_index)
-   [Teksten op B1-niveau? Liever direct duidelijk! – Gebruiker
