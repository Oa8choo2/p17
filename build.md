# build


## Current build process

::: incremental
1.  markdown (md) files stored in git repo at github
2.  some config files
3.  some bash
4.  some python (mainly to create OS and site versions)
5.  some Jinja2
6.  **mkdocs** to convert md files to html
7.  some mkdocs plugins
8.  mkdocs-material for theming and some extras
9.  build on github, result in github pages
:::
